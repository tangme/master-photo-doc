# m-pencil-tool

## Attributes
| 参数      | 说明     | 类型  | 可选值 | 默认值                                                       |
| --------- | -------- | ----- | ------ | ------------------------------------------------------------ |
| size      | 画笔大小 | Array | -      | [2, 4, 6]                                                    |
| predefine | 画笔颜色 | Array | -      | ["#ff4500","#ff8c00","#ffd700","#90ee90","#00ced1","#1e90ff","#c71585"] |
