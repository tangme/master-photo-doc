# m-view

## Attributes
| 参数     | 说明                                               | 类型     | 可选值 | 默认值   |
| -------- | -------------------------------------------------- | -------- | ------ | -------- |
| src      | 默认图片                                           | *String* | -      | -        |
| width    | 组件宽度，须提供单位 `px、%、`等                   | *String* | -      | 100%     |
| height   | 组件高度，须提供单位 `px、%、`等 PS:包涵工具栏高度 | *String* | -      | 100%     |
| bg-color | 默认背景颜色                                       | *String* | -      | \#1f2430 |

## Slot
| 名称   | 说明                  |
| ------ | --------------------- |
| header | 在canvas 上方插入内容 |
| footer | 在canvas 下方插入内容 |
