# m-toolbar

## Attributes
| 参数  | 说明                         | 类型    | 可选值                                              | 默认值 |
| ----- | ---------------------------- | ------- | --------------------------------------------------- | ------ |
| tools | 配置工具栏按钮，默认启用所有 | *Array* | rect/ellipse/arrowline/pencil/mosaic/text/undo/file | -      |

## Slot
| 名称   | 说明                  |
| ------ | --------------------- |
| header | 在canvas 上方插入内容 |
| footer | 在canvas 下方插入内容 |
